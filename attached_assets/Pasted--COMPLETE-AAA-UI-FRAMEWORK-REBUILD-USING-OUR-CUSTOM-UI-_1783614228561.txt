# 🎮 COMPLETE AAA UI FRAMEWORK REBUILD USING OUR CUSTOM UI KITS

## OBJECTIVE

The current user interface is no longer acceptable.

It contains generic UI elements that do not match the premium visual quality of the game worlds.

Our newly designed UI Kits are now the OFFICIAL DESIGN SYSTEM of the entire game.

The goal is NOT to modify the current UI.

The goal is to completely remove the existing interface and rebuild the entire UI framework using our handcrafted UI Kits.

Every menu, every button, every panel, every popup, every scoreboard, every icon and every result screen should come exclusively from our custom UI assets.

This should feel like a professionally built AAA mobile game.

# IMPORTANT IMPLEMENTATION RULE

The Bamboo Grove UI Kit is the MASTER REFERENCE.

The UI layout, spacing, animations, touch areas, and interactions implemented for Bamboo Grove become the template for every other theme.

Moon Shrine, Crimson Temple, Imperial Heaven Palace, and Dojo Gate must reuse the exact same framework, changing ONLY their visual assets, colors, decorative motifs, particles, and sounds.

Do NOT create separate UI logic for each theme.

--------------------------------------------------

# FINAL REQUIREMENT

The player should feel that each world has its own handcrafted interface while the game behaves consistently everywhere.

When switching between Bamboo Grove, Moon Shrine, Crimson Temple, Imperial Heaven Palace, and Dojo Gate, the UI should transform seamlessly into the selected theme without changing layout, controls, or usability.

The final result should look like a commercially released AAA mobile game, where every UI component feels handcrafted, polished, highly responsive, visually immersive, and perfectly integrated with its corresponding game world.
