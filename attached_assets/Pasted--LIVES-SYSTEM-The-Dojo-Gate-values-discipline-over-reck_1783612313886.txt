# LIVES SYSTEM

The Dojo Gate values discipline over recklessness.

Players begin each game with **3 Samurai Crests** displayed at the top of the screen instead of traditional hearts.

Each Samurai Crest represents the player's honor and focus.

Life Rules:

• Start every game with 3 Samurai Crests.
• Missing 1 fruit loses 1 Samurai Crest.
• Missing 3 fruits results in Game Over.
• The loss of a life should trigger:
  • Small screen shake
  • Brief red flash
  • Wooden strike sound
  • One Samurai Crest breaking into particles

The Samurai Crest UI should match the ancient Japanese dojo aesthetic using gold, bronze, and black accents.

---

# CURSED ONI MASK RULE

The Cursed Oni Mask is NOT a normal bomb.

It contains sealed demonic energy.

If the player slices the Oni Mask:

• The demon seal instantly breaks.
• Dark spiritual energy erupts.
• Black smoke fills the screen.
• Crimson spirit particles explode outward.
• The game immediately ends regardless of remaining lives.
• No second chances.
• No shield can block it.

This rule emphasizes precision and discipline.

# DIFFICULTY PROGRESSION

The challenge should increase naturally.

Early Game

• Slower fruit speed
• Lower launch height
• 3–5 fruits

Mid Game

• Faster launches
• More simultaneous fruits
• Higher combo opportunities

Late Game

• 6–8 fruits simultaneously
• Faster trajectories
• Mixed fruit patterns
• Higher precision required

The game should never feel unfair.

Difficulty should reward skill rather than randomness.

# POWER-UPS

Occasionally spawn legendary dojo relics.

🌸 Zen Focus

• Slows time for 7 seconds.
• Fruits move slower.
• Sword remains full speed.

---

⚔️ Blade Mastery

• Increases sword slash width by 20%.
• Improves slice accuracy.
• Lasts 10 seconds.

---

🌺 Cherry Blossom Fury

• One successful slash automatically slices every fruit currently on screen.
• Beautiful cherry blossom explosion.
• Oni Mask is NOT affected.

---

🔔 Grandmaster's Bell

• Protects against ONE missed fruit.
• Does NOT protect against the Oni Mask.
• Bell disappears after activation.

Power-up spawn chance:

Approximately once every 45–60 seconds.

Never spawn multiple power-ups simultaneously.

# FINAL EXPERIENCE

The Dojo Gate should be the most refined and disciplined game zone in the entire game.

Every element—from the Grandmaster's Bell Katana and sacred fruits to the Cursed Oni Mask and serene environment—should embody precision, patience, and mastery.
